# Car Line Follower





## Description
> The car is following only the black path. 


## Getting Started

### Components 
* Raspberry pi zero W
* L298n motor driver
* KY-033 line tracking sensor
* Rechargeable battery
* 4WD robot chassis

### Dependencies
* raspberry pi IOS
* Python3
